При всички порядъци да се има предвид, че константите са игнорирани навсякъде.
1. For цикълa се изпълнява n пъти, като на всяка итерация изпълняваме While цикъла n пъти (всеки път разликата end - start намалява с 1, докато не стане 0) => O(n*n)
2. O(n-x + x*m), където x е броя четни числа в първата колона на matrix. For цикълa се изпълнява n пъти, а във вътрешния For цикъл се влиза x пъти.
   Ако x е константа => О(n + m), а ако x е от порядъка на n => O(n*m)
3. O(n^2) - заради рекурсията, но все пак да се има предвид, че говорим за квадратна матрица
